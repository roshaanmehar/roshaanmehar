## About

I'm a problem-solving focused developer with a strong foundation in full-stack development and data-driven applications. My journey started with web fundamentals (HTML, CSS, PHP), deepened with CS50 and data structures, then expanded into the MERN stack, NASA Space Apps Challenge, and a lasting interest in web scraping and automation. I enjoy turning complex problems into clean solutions—from Rubik's cube solvers to AI-powered product matching and time-tracking apps.

**Highlights:**
- Full-stack: Next.js, React, Node, Express, Python (Flask/FastAPI)
- Data & automation: scraping, OpenCV, vector DBs, habitability/analytics pipelines
- Delivered: image-based cube solver, competitive intelligence platform, NASA hackathon project, retro time tracker
